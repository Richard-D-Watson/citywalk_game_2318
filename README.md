# City Walk 闯关（静态网页项目）

## 功能
- 主页面：路线总览（闯关时间线）+ 总进度（星星）
- 关卡预览：Bottom Sheet（抽屉）展示封面/描述/解锁状态
- 关卡页：多任务（填空/选择/上传），任务完成得星
- 解锁规则：完成上一关任意一个任务，即可解锁下一关
- **挑战/奖励页**：若某关包含“挑战/奖励”任务，完成该关全部主任务后会自动打开新页面显示挑战/奖励，并可点击“我已完成/领取”
- 总结页：用时、总星、称号、回忆卡（含上传照片预览）+ 挑战/奖励完成统计
- 本地保存：localStorage（刷新不丢进度）

## 运行方式（推荐）
在本目录下运行：
- Python 3：`python -m http.server 8000`
然后浏览器打开：`http://localhost:8000`
